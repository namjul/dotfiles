Aspects are a focal point that attracts attention, organizes surrounding elements, and gains strength through interaction. They all run idempotently.

## Navigation

```
aspects/
├── aur/           Arch — system packages, firewall, firmware
├── dotfiles/      home directory symlinks and templates
├── fonts/         Font setup
├── homebrew/      Homebrew package management
├── i3status-rust/ i3 status bar
├── interception/  keyboard remapping (caps → esc/ctrl)
├── meta/          fig framework verification
├── nala/          Ubuntu — apt package installation
├── nvim/          Neovim configuration
├── server/        remote server provisioning (hobl.at)
├── shell/         login shell (fish)
├── ssh/           SSH config and keys
└── systemd/       Linux — systemd user services
```
